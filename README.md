# 🌊 漂流瓶小游戏 - Drift Bottle Fortune

> 在 GitHub 的海洋里，捡起一个漂流瓶，收获今日运势、签文或一句温暖的话

[![漂流瓶运势生成器](https://github.com/lly20050425/drift-bottle-fortune/actions/workflows/fortune-generator.yml/badge.svg)](https://github.com/lly20050425/drift-bottle-fortune/actions/workflows/fortune-generator.yml)

## 🎮 玩法说明

### 🧭 捡瓶子
浏览 [Issues 列表](https://github.com/lly20050425/drift-bottle-fortune/issues)，每个 Issue 就是一个漂流瓶！点击打开，看看里面藏着什么运势或温暖的话语。

### 📝 扔瓶子
创建一个 [新 Issue](https://github.com/lly20050425/drift-bottle-fortune/issues/new)，写下你想对陌生人说的话、你的运势预测、或者一段鼓励的话，然后贴上 `漂流瓶` 标签。

### 💬 回信
在别人的漂流瓶下评论，传递温暖和祝福。

### ⭐ 收藏
给你喜欢的漂流瓶点个 Reaction（👍 ❤️ 🎉），让更多人看到。

## 🎴 漂流瓶类型

| 类型 | 标签 | 说明 |
|------|------|------|
| 🔮 每日运势 | `daily` | 综合运势、幸运数字、幸运颜色、今日锦囊 |
| 🃏 签文 | `tarot` | 塔罗牌签文，含正位/逆位和签运 |
| 💌 温暖寄语 | `encourage` | 一句温暖的话，来自匿名的好心人 |
| 🐲 生肖运势 | `zodiac` | 十二生肖运势，含爱情/事业/财运评分 |
| 🍀 幸运清单 | `lucky` | 幸运数字、幸运色彩、幸运美食、幸运任务 |

## ⚙️ 自动化

本项目使用 GitHub Actions 每天自动生成一个随机类型的漂流瓶（北京时间每天 08:00）。

你也可以手动触发：前往 [Actions 页面](https://github.com/lly20050425/drift-bottle-fortune/actions/workflows/fortune-generator.yml)，点击 "Run workflow"。

## 📜 扔瓶子的格式

创建 Issue 时，请使用以下格式：

```
标题：[类型标签] 你的标题
标签：漂流瓶 + 对应类型标签

正文：
## 🌊 我的漂流瓶

> 写下你想说的话...

---
*来自一位匿名的好心人*
```
